Up 
post /createContacts
(createContactsPost)
creates a new contact from the contact list database

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json
  • application/xml

Request body

body contact (required)
Body Parameter —

Responses

200

successfully created contact.

404

unsuccessful creation of the contact.
Up 
get /GetAllContacts
(getAllContactsGet)
obtains all the contacts belonging to a specific user in the database

Query parameters

userId (required)
Query Parameter — The id of the contact that is used for partitioning may be similar for more than one contact

Return type

contacts

Example data

Content-Type: application/json
[ {
  "surname" : "Willard",
  "name" : "Samuel",
  "phonenumber" : "713276358",
  "id" : "peter123",
  "userId" : "fever2020"
}, {
  "surname" : "Willard",
  "name" : "Samuel",
  "phonenumber" : "713276358",
  "id" : "peter123",
  "userId" : "fever2020"
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

successfully pull all user contact information contacts

404

unsuccessful contacts retrieval
